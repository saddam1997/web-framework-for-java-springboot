Hello, !

This is a starter project with Spring Boot, Thymeleaf, MySQL, RESTful API, OAuth2, Bootstrap, AngularJS and AngularJS Material.

Go to Admin Panel

Features

User & Role

Create your admin user and manage registered users and roles.

Menu Management

Custom your admin menus for each roles via existing templates.

App Management

Register external applications for using SSO and OAuth features.

SSO & OAuth

Login and receive your account information according to OAuth standards.

International

Multiple languages support and which is defined in i18n folder.

CMS

Basic Content Management System. Supports pages, articles and attachments.

Full Text Search

CMS has been implemented Full Text Search feature. This feature is based on Apache Lucene.

Themes

Support 10 themes for Admin Panel based on AdminLTE.
